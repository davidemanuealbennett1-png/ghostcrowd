How it works

Three steps from idea
to insight.

No 3D models. No CAD files. No engineering degree. Just draw and simulate.

01

Sketch your space

Draw walls, place tables, mark doors. Use a template or start from scratch. Grid snapping keeps everything clean.

02

Run the simulation

Set how many people and hit run. The Social Force Model — the same physics used by airports and stadiums — calculates every movement in real time.

03

See where crowds form. Toggle the heat map to find density hotspots. Yellow markers flag exact bottleneck zones. Then adjust and rerun.

The science

Enterprise-grade physics. Consumer-grade simplicity.

GhostCrowd uses the Social Force Model (Helbing & Molnár, 1995) — the same algorithm used by MassMotion, LEGION, and AnyLogic — tools that cost $10,000–$50,000/year and require CAD expertise.

We took the physics and wrapped it in something anyone can use in 60 seconds.

# Social Force Model — each agent feels:

F_desired = pull toward destination
F_social = repulsion from nearby people
F_wall = repulsion from walls & obstacles
F_noise = random human variation

# Result: realistic emergent crowd behavior

simulate(agents=200, dt=0.05s)
# → 151 simulation steps/second
# → Heat maps, bottleneck detection
# → Exportable PNG reports
